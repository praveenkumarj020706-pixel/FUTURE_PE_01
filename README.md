# Task 1: AI Website Content Generator

## 🔹 Objective:
To create professional and human-like website content for different types of businesses using AI tools.

---

## 🔹 Tools Used:
- ChatGPT  
- Google Docs  
- GitHub  

---

## 🔹 Skills Gained:
- Prompt Engineering  
- Website Copywriting  
- Business Content Creation  
- Customer-focused Writing  

---

## 🔹 Key Features:
✔ Homepage content with clear messaging  
✔ Service-based structure  
✔ Strong call-to-action (CTA)  
✔ Simple and engaging tone  
✔ Adaptable for different business types  

---

## 🔹 Description:
In this task, AI was used to generate website content for two different business types — an Online Fitness Coaching platform and a Digital Marketing Agency.

The content includes headline, introduction, services, and CTA, making it suitable for real-world website usage.

The focus was on keeping the content simple, clear, and engaging for users.
